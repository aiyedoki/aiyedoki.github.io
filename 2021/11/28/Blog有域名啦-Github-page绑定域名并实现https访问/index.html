Blog有域名啦~ Github page绑定域名并实现https访问

  • 2021-11-28
  • 作者 黑川秋原
  • 607 字
  • 次阅读
本文最后编辑于 前，其中的内容可能需要更新。

​ 附上本站域名😋 https://kidoki.xyz/

​ 今晚才购买的xyz域名，实名验证半个小时就通过了，并且能够正常解析，没有提醒让我备案。可能是因为我一年前在阿里云购买域名的时候已经做过备案了，不过那个域名已经过期了。

​ 其实我对这些解析，还有云服务器的操作都不太理解hhh ， 每次需要操作就上网搜索，照葫芦画瓢，总能解决问题。比如这次的域名解析， “www、CANME、TTL“ 这些都不太明白是什么意思，不过借助平常积累的经验， 还是能够知道这些东西该怎么填怎么用，不过这些名字具体是什么东西 我还真不太懂

​ 就好像我会踩油门，但是不知道油门是靠什么原理让车动起来的，发动机里的东西我就完全不懂了！奇怪的比喻增加了(￣﹃￣)

​ 其实域名解析到Github page也就分为两大步吧 ，但还是记录下，免得自己以后又忘记了，大佬们别笑话

  1. 进入阿里云的解析设置 直接抄！我记得腾讯云的面板也是一样的设置，只是UI不同

  2. 到Blog的Github仓库 设置页，找到Pages，填写域名

​ 到这就完成了，又水了一篇文章:-D

​ 不过还是留下了一个问题：在地址栏直接输入kidoki.xyz，网页前缀有几率不会跳转到https，而是http。以后再想想怎么解决吧:-D
